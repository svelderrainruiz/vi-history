# Imported SyRS: Runtime Settings CLI Terminal Entrypoint

## System Context

This public MIT slice defines the installed-user bare `vihs` terminal
entrypoint contract.

The first admitted unit is limited to materialized-entrypoint facts, supported
terminal-session admission state, runtime lookup/recovery facts, and
stale-or-missing launcher failure guidance. It does not start raw terminal
prompt loops, compare execution, LabVIEWCLI execution, Docker execution or
orchestration, proof-out expansion, live-session proof, packaging, Marketplace
publication, or source copying.

## System Need

Installed users need `vihs` to be a discoverable terminal command without
hidden path reconstruction, profile editing, admin elevation, machine-wide
installation doctrine, or a separate prebuilt CLI payload. Maintainers need
that command surface specified as public clean-room behavior before any process
prompt loop or runtime execution is implemented.

## Public Boundary

This import carries requirements only. It does not carry implementation source,
private evidence, release credentials, private issue templates, private
control-plane instructions, or publication automation.
